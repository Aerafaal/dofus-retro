# Dofus Retro

> An emulator that I use as a POC to reverse engineer the Dofus Retro game in order to better understand how it works

## Requirements

- [.NET7][dotnet]
- [PosgreSQL][pgsql]
- [Rider][rider]

## Setup

```sh
dotnet restore
dotnet build
dotnet run
```
